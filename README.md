# VARIABLES
## 1.0
- 'main' method가 존재해야지 실행
- ';'는 필수이다.
    
## 1.1
- 변수를 선언할 때는 ver를 사용
- var은 변수의 타입을 구체화 ❌
    - 하지만 변수를 업데이트 할 시, 변수의 본래 타입과 일치해야 함
    - 변수를 선언할 때는 명시적으로 변수의 타입을 지정
## 1.2
- dynamic 타입인 경우, 어떤한 타입으로 지정이 가능
## 1.3
- null safety은 개발자가 null 값을 참조할 수 없도록 하는 기능
- 코드에서 null값을 참조하면 런타임 에러가 발생        - 하지만 아래와 같이 작성이 가능

```
        {변수형}? {변수이름} = '{변수값}'
        {변수 이름} = null;
```
